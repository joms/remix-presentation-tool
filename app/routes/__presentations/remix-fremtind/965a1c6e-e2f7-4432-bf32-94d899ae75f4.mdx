## Hva om vi heller bare bruker APIet?

```tsx
export const action = () => {
    const session = await storage.getSession();

    session.set("username", username);

    // Response status: 302, headers: { "Set-Cookie": "..." }
    return redirect("/", {
        headers: {
            "Set-Cookie": await storage.commitSession(session),
        },
    });
};
```
